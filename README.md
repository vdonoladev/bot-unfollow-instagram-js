# bot-unfollow-instagram-js
<h3>Português</h3>

Bot (Instagram) para deixar de seguir usuários automaticamente

Modo de uso:

- Encontre as pessoas que você quer deixar de seguir
- Abra o Inspecionar
- Vá em Console
- Cole o código que está dentro do arquivo "index.js" e dê Enter
